## 完整操作步骤（请按顺序执行）

### 步骤 1：重新创建飞书机器人

1. 打开飞书群聊
2. 点击右上角 "..." 或群设置
3. 选择 "机器人管理"
4. **删除**当前的机器人
5. 点击 "添加机器人" → "自定义机器人"

### 步骤 2：配置机器人信息

填写以下信息：
- **机器人名称**：访客留言机器人
- **机器人描述**：接收网站访客留言

### 步骤 3：开启签名校验（最重要！）

1. 创建机器人后，会显示"安全设置"页面
2. **必须勾选"签名校验"选项** ✅
3. 勾选后，系统会**自动生成一个32位的密钥**
4. 密钥格式类似：`ONHtg1cudF3QKiJ8edyrce0123456789`（**32个字符**）

### 步骤 4：复制完整密钥

1. 找到密钥旁边的"复制"按钮 📋
2. **点击复制按钮**（不要手动选择复制）
3. 确保复制的是完整的 32 个字符

### 步骤 5：更新配置文件

1. 打开项目根目录的 `.env` 文件
2. 找到 `FEISHU_WEBHOOK_SECRET` 这一行
3. 替换为新的完整 32 位密钥

**修改前（错误）：**
```env
FEISHU_WEBHOOK_SECRET=ONHtg1cudF3QKiJ8edyrce
```

**修改后（正确）：**
```env
FEISHU_WEBHOOK_SECRET=ONHtg1cudF3QKiJ8edyrce0123456789
```

### 步骤 6：验证配置

1. 保存 `.env` 文件
2. 运行诊断脚本：
   ```bash
   npx tsx scripts/diagnose-feishu.ts
   ```
3. 确认显示 "✅ 所有检查通过！"

### 步骤 7：测试功能

## 常见问题解答

### Q: 为什么我的密钥只有22个字符？
**A:** 因为您在创建机器人时没有勾选"签名校验"选项，或者复制时没有复制完整。

### Q: 开启签名校验后，密钥在哪里？
**A:** 在机器人的"安全设置"页面，勾选"签名校验"后会自动显示32位密钥。

### Q: 我可以手动添加缺失的10个字符吗？
**A:** **绝对不可以！** 密钥是系统自动生成的，不能手动修改。必须重新复制完整的32位密钥。

### Q: 复制后还是不够32字符怎么办？
**A:**
1. 确保使用"复制"按钮，不是手动选择
2. 检查是否包含了空格或换行
3. 重新创建机器人，确保开启签名校验
4. 使用复制按钮完整复制

### Q: 密钥格式是什么样的？
**A:** 32个字符的字母数字组合，例如：
```
ONHtg1cudF3QKiJ8edyrce0123456789
```

### Q: 如何验证密钥是32个字符？
**A:** 运行诊断脚本：
```bash
npx tsx scripts/diagnose-feishu.ts
```
或者手动数一下字符数量。
